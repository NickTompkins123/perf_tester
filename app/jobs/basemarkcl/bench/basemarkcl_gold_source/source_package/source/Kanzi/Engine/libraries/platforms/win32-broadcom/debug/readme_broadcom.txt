This directory is for libraries for the Broadcom device.

The driver works only with Windows XP. You can run the device in Windows 7 using the Virtual XP.

Usage: link the application with khclient.lib. The library contains EGL, VG, GL ES 1.1 and GL ES 2.0 implementations. 

Before running the application run the khbridge.exe program and start the VMcs_khronos_smallint.bin application on the Broadcom board.
